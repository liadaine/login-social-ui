# 🔐 Login com Redes Sociais

Projeto desenvolvido com HTML e CSS criando um formulário de login com opção de autenticação via redes sociais, com foco em layout moderno e organização visual.

---

## 🛠️ Tecnologias utilizadas
- HTML
- CSS

---

## 🎯 Objetivo
Praticar a criação de formulários mais completos, incluindo opções de login com redes sociais e organização de elementos na interface.

## 📚 O que foi praticado
- Estruturação de formulário em HTML  
- Estilização com CSS

- Alinhamento e posicionamento de elementos
